.SH NAME
ArchiveManager - high level program for making various operations based on archives.
.SH SYNOPSIS
BigScript [-h]
.SH DESCRIPTION
ArchiveManager provides a few basic operations using very comfortable zenity interface:
 - creating new archive from selected files and directories
 - looking inside the archives
 - unpacking the archives in selected directories
 - deleteing files from selected archives
.SH OPTIONS
-h  -  switches on the manual page for script
